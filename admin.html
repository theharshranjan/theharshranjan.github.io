Forgot password? Recovery guide

🔐 Password Recovery — 3 Ways

Method 1 — Open the HTML file in Notepad/VSCode and search for ADMIN_PASSWORD. You will see your password there. Change it anytime.

Method 2 — Emergency bypass: Open this page, press F12 → Console tab, type exactly:
sessionStorage.setItem('samarpan_admin_auth','1');location.reload();
and press Enter. This logs you in for one session.

Method 3 — Reset password: In the HTML file find the line that says:
const ADMIN_PASSWORD = '...';
Change it to any new password you want and save.

💡 Tip: Save your password in Google Keep or Notes app for safety.
