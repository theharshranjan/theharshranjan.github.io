Setup & Configuration Guide

Firebase Auth, Firestore rules, email notifications, and phone installation.

Step 1 — Create Firebase Admin AccountRequired
1. Go to Firebase Console → Authentication → Users
2. Click "Add user" → enter email + password → click "Add user"
3. Use these credentials to sign in here.

Forgot password? Use the "Forgot password?" link on the login screen.
Step 2 — Firestore RulesRequired

Go to Firebase Console → Firestore → Rules and paste:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /homepage_leads/{docId} {
      allow create: if isValidLead();
      allow read, update, delete: if isAdmin();
    }
    match /homepage_newsletter/{docId} {
      allow create: if isValidEmail();
      allow read, update, delete: if isAdmin();
    }
    match /homepage_quiz_results/{docId} {
      allow create: if true;
      allow read, update, delete: if isAdmin();
    }
    match /homepage_visitors/{docId} {
      allow create: if true;
      allow read, update, delete: if isAdmin();
    }
    match /homepage_yoga_sessions/{docId} {
      allow create: if isValidEmail();
      allow read, update, delete: if isAdmin();
    }
    match /homepage_referrals/{docId} {
      allow create: if isValidEmail();
      allow read, update, delete: if isAdmin();
    }
    match /homepage_bookings/{docId} {
      allow create: if isValidLead();
      allow read, update, delete: if isAdmin();
    }
    match /homepage_about_leads/{docId} {
      allow create: if isValidLead();
      allow read, update, delete: if isAdmin();
    }
    match /samarpan_journal_posts/{docId} {
      allow read: if true;
      allow create, update, delete: if isAdmin();
    }
    match /{document=**} {
      allow read, write: if false;
    }
    function isAdmin() {
      return request.auth != null;
    }
    function isValidEmail() {
      return request.resource.data.keys().hasAny(['email'])
        && request.resource.data.email is string
        && request.resource.data.email.size() > 4
        && request.resource.data.email.size() < 200;
    }
    function isValidLead() {
      return request.resource.data.keys().hasAll(['email', 'name'])
        && request.resource.data.email is string
        && request.resource.data.name is string
        && request.resource.data.email.size() > 4
        && request.resource.data.name.size() > 0;
    }
  }
}
Step 3 — Email Alerts for New Leads (Free)Optional
Get email alerts when new leads come in using EmailJS — free up to 200 emails/month.
1. Sign up free at emailjs.com
2. Create a service (Gmail works) and a template
3. Enter your keys in Settings → Email Notifications
4. Test with the "Send Test" button
Step 4 — Install on Phone (PWA)Optional
📱 Android (Chrome): Tap ⋮ menu → "Add to Home screen"
🍎 iPhone (Safari): Tap Share → "Add to Home Screen"
